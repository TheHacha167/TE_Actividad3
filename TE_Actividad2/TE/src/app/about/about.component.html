Sobre la Aplicación

Esta aplicación permite encontrar y filtrar gasolineras en España según la ubicación del usuario.

Objetivos del Proyecto

  • Mostrar las gasolineras más cercanas según la ubicación del usuario.
  • Permitir filtros por empresa, tipo de carburante y ubicación.
  • Brindar información clara sobre los precios del combustible en tiempo real.

Tecnologías Utilizadas

  • Angular para el frontend.
  • API REST para obtener datos de gasolineras.
  • Geolocalización para calcular distancias.

Equipo de Desarrollo
